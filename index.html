# Explain var,let and const

1. var :- var is a global scope or function scope variable.
we can excess it from enyware in the code.
it get allocated memory in global space.

[var variable can updated and re-declared.]

Ex:- var a=5;
console.log(a)
ANS= 5

2. let:- let is block scope variable
we cannot excess let variable outsitde the block.
let get allocated memory in temporar dead zon.

[let variable can be updated but not re-declared.]

Ex:-{ let b=10;
console.log(b)
}
ANS=10

3.const:- let is block scope variable
we cannot excess const variable outsitde the block.
const get allocated memory in temporar dead zon.

[const variable can neither be updated nor re-declared.]

Ex:-{ const c=20;
console.log(c)
}
ANS=10

# Hoisting

  • In js each and every variable declared using either var let or const get some memory space.
  • if variable is declared using var then it will get memory space in global space so if you will try to excess the variable in this case before initialization it. then it will return undefined.
  • if variable is declared using either let or const then it will get memory space in separate space not in global space. so if you will try to xcess the variable in this case before initializing it then it will return reference error saying you cannot access the variable befor initialization.
